# Greek Real Estate Document Templates

This folder contains downloadable DOCX templates for common Greek real estate documentation.

## Available Templates

1. **brokerage-mandate.docx** - Εντολή Ανάθεσης Μεσιτείας
   - Authorization form for real estate agent to sell or rent property
   
2. **lease-agreement.docx** - Ιδιωτικό Συμφωνητικό Μίσθωσης
   - Standard residential rental contract
   
3. **handover-protocol.docx** - Πρωτόκολλο Παράδοσης-Παραλαβής
   - Property handover checklist with meter readings
   
4. **viewing-confirmation.docx** - Βεβαίωση Επίσκεψης Ακινήτου
   - Confirmation of property viewing through agency

## Creating Custom Templates

To create custom DOCX templates:

1. Open Microsoft Word or Google Docs
2. Use the placeholder variables in double curly braces: `{{placeholder_name}}`
3. Save as .docx format
4. Place in this folder

## Placeholder Variables

See `/lib/templates/template-definitions.ts` for all available placeholder variables for each template type.
